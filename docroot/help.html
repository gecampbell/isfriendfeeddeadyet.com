Is FriendFeed dead yet?

How this works

This site verifies that FriendFeed is up and operational and producing valid content. It analyzes the HTTP response status code as well as the size of the returned page to determine if FriendFeed is alive or not. Using our patent-pending "death checker" algorithm, we can determine with a high degree of accuracy whether or not the two people still using FriendFeed are going to be inconvenienced.

What else can you do with it?

You can also verify whether or not a particular FriendFeed user is still alive:

http://isfriendfeeddeadyet.com/username

For example, to check to see if my account is still alive:

http://isfriendfeeddeadyet.com/glenc

If you have any further questions or comments, feel free to leave me a message at my FriendFeed account. Unless it's dead, of course.
